Detailed Description

Base interface for visualisation component.

It is a handle class ("smart" pointer). IVtk_Interface is a base class for OCC interfaces for VTK classes, such as interface for shape objects (IVtk_IShape) and view(IVtk_IView), shape triangulation (IVtk_IShapeData, IVtk_IShapeMesher) and picking (IVtk_IShapePickerAlgo).
